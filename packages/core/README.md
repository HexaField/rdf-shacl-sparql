# @template/core

Core library for the Agent-Centric P2P Stack. This library provides the fundamental building blocks for creating decentralized, local-first applications strictly adhering to the "Library" architecture.

## Architecture

- **Identity**: DID management and resolution.
- **Store**: RDF Quad Store (The Card Catalog).
- **Validation**: SHACL (The Rulebook).
- **Query**: SPARQL + ZCAP (The Librarian).
- **Visuals**: VC 2.0 (The Verified Entries).
